# HTML
- HTML is used to create structure(Layout) of webpages.

# CSS
- CSS is used to apply the style for HTML pages.

# Bootstrap : -
- Bootstrap is a libary of HTML and CSS which is used to create fast Responsive Website.

# JavaScript : -
- It's a high level programming language which is used to create Dynamic functionally on web pages.

## EX :-
```
<!DOCTYPE html>
<html lang="en">
<head>
 <title>20-Jan-2026</title>
</head>
<body>
    Hello World , I am Subhadeep !
    <br>
    <button>Read More</button>
</body>
</html>
```

## HTML DOCTYPE Declearation : -
- Doctype declaration is used to represent the version of HTML.
- The latest version of HTML is HTML5.
- In HTML File the Doctype declearation is '<!DOCTYPE>'.
