
# <b> Tag </b>

## Syntax : -
```
<tagname>  -----   </tagname>
```

# HTML Attributes : -
- By using Attributes we can provide aditional functionality of HTML pages.

## Syntax : -
```
<tagname  attname = "value" >
        content
        -----  
</tagname>
```

## EX :-
```
<h1 align = "center">
    Hello World
</h1>
```

# Marquee  Tag : -
```
<marquee scrollamount = "50px" behavior = "alternative" direction = "right" loop ="4">
    Subhadeep Ghosh
</marquee>
```
