###  <b> Anchor Tag</b>
- Anchor tag is used to connect from one document to another document like HTML page , Web Pages.

- Link is a text,which contents the references of another webpages.

- link is also known as hyperlink or hypertext.

#### Syntex : 
```
    <a href="url">
        Content 
    </a>
```    

## Ex: 
     <a href="https://www.amazon.in/">Subhadeep Ghosh</a>

### <b> Img tag</b>

#### Syntex :

    <img src="./download.png"
