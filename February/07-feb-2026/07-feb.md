# <b> Loop </b>

- 5 Loop is Available in javascript
```
    1.  For
    2.  While
    3.  Do-While
    4.  For in Loop
    5.  For of Loop
```

```

}
```

## Break :- 
```
Break is a flow control statement which is used to terminate immidiately the execution of switch case or loops.
```

## Return :- 
```
Return statement is used to turminate the function.
```

## Continue :- 
```
Continue is a loop control statement which is used to skip the current itaration of loop and proceed to next itaration immidiately.
```
